what is HD-Insights
Azure HDInsight is a cloud-based service provided by Microsoft Azure that simplifies the deployment, management, and scaling of big data processing solutions. It is based on popular open-source frameworks such as Apache Hadoop, Apache Spark, Apache Hive, Apache HBase, and others.
